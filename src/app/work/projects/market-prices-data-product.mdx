## Objectif

Suivre l’évolution des prix de marché et produire un dataset propre pour le reporting.

## Pipeline

- Génération d’un jeu de données simulé
- Nettoyage et standardisation
- Calcul d’indicateurs de qualité des données
- Dataset final enrichi pour le dashboard

## Analyse IA

L’application propose une analyse économique assistée (optionnelle) via un modèle IA.
